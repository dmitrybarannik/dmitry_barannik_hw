Блок навигации

  • Ширина 15em(240px)
  • Должен быть цвет фона
  • Слева
  • Блок навигации наполнить контентом достаточным, чтобы по высоте он начинал захватывать вторую статью.

Статья 1

Размер блока:
Должен занимать всю доступную ширину.
Должен присутствовать цвет фона.
Во всех "статьях" сделать отступ снизу от соседей равным .2em, при помощи свойства margin-bottom.

Статья 2

Размер блока: Занимает всю доступную ширину.
Должен присутствовать цвет фона.
Вс всех "статьях" сделать отступ снизу от соседей равным 2em, при помощи свойства margin-bottom.
Картинка вставлена при помощи тега, плавание организовано по правому краю внитри данной статьи для этой картинки.

Нижний блок. Высота по контенту. Дерби «Динамо» и «Минска» решил лишь гол: классная пятка и топ-дуга от Бахара (праздновать не стал)
